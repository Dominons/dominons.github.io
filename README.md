**CONTEXTO DEL PROYECTO (DANIFU777 - Dominon & DEEP FLOW)**
- **Arquitectura:** SPA (Single Page Application) serverless basada en HTML5, CSS3 aislado y Vanilla JavaScript modular.
- **Persistencia:** Enfoque "Local-First" usando `localStorage` para almacenamiento local inmediato y Firebase (Auth, Firestore, Realtime Database) para sincronización en la nube (según plan).
- **Módulos principales:**
  1. **Dominon:** Dashboard de productividad. Incluye Reloj/Fecha, iframe interactivo de TradingView con selector de activos, Bloc de notas dinámico con pestañas, Calculadora aritmética, Temporizador Pomodoro con personalizador, y controlador unificado de playlist de YouTube con sincronización de progreso en RTDB (intervalo de 10s).
  2. **DEEP FLOW:** Terminal simulador de flujo cripto. Lector en tiempo real de API de CoinGecko (BTC, ETH, Gas), generador de ledger on-chain simulado (ballenas, liquidaciones, swaps) y Binance (volumen spot, prueba de reservas, orden book). Sintetizador de audio nativo con Web Audio API y soporte global multilingüe (ES/EN).
  3. **QUANTIS:** Lienzo de trabajo multimodal con paneo/zoom 3D. Creación de bloques de texto interactivos con autodetección de URLs, importación local de imágenes/PDF en Base64, sistema de proyectos/páginas en barra lateral, y capa de dibujo vectorial (pincel/borrador Premium).
- **Monetización (3 Planes):** Normal (Límite 3 notas, Quantis 2 proy/3 pág, sin dibujo, local), Extended (Límite 5 notas, Quantis 4 proy/7 pág, exportar/importar, local), Premium (Límite 10 notas en nube, Quantis 10 proy/20 pág, dibujo desbloqueado).
